{{section: ⚙️ Configuration}}

All settings live in ``config/windowed-guis/`` and can be edited in-game from the gear icon on the taskbar. The folder holds:

* ``client.json`` — client-side appearance and behavior options
* ``server.json`` — server-side options such as the auto-close distance
* ``windows.json`` — saved window positions, scales, and tab colors
* ``blacklist.json`` — screens that should stay full-screen
* ``previews/`` — cached window previews

> 💡 **Tip:** edits made to the config file apply the next time you load a world.
